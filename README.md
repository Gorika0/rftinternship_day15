# rftinternship_day15

A simple Python Exploratory Data Analysis (EDA) dashboard project using Matplotlib and Pandas.
This project demonstrates multiple plots, trend analysis, comparison charts, distribution analysis, and visual outlier detection using subplots.

Features
Line Chart (Trend Analysis)
Bar Chart (Comparison)
Histogram (Distribution)
Scatter Plot (Outlier Detection)
Subplots Dashboard
Insight Generation

Charts Used
1. Line Chart
Shows the sales trend over different months.
2. Bar Chart
Compares profit values for each month.
3. Histogram
Displays the distribution of sales values.
4. Scatter Plot
Used for visual outlier detection.

Outlier Detection
The project visually detects an outlier using a scatter plot.
July sales value = 500
This value is much higher than other months
Therefore, it is marked as an outlier
